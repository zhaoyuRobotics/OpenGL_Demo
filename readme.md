### 本示例是什么: ### 
本示例读取3ds文件，bmp纹理图片，然后加载在Opengl中显示  
  
  ------------------------------------------------------  
  
### 环境及运行方式: ###  
linux14.04  32bit  opengl2.1  
make 后可直接运行可执行程序： ./tutorial4
